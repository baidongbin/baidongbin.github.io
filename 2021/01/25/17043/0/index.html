选择相应版本代码下载 zip 到本地，本文使用 7.8.0 版本

安装有两种方式，一种直接 github 克隆项目，方便版本升级，一种直接下载文件，本文选择直接下载，因为需要修改源代码，克隆意义不大。

解压下载的 zip 将文件拷贝到项目的 themes 文件夹中，并重命名文件为 next，如图

主题配置文件选择

配置文件有两个，一个是项目的 _config.yml，一个是主题的 _config.yml。

Hexo 5.0.0 起提供独立的 _config.[theme].yml 文件的方式，但是不支持热部署。

next 主题也提供了两个解决方案，可以参见next包中的 docs/zh-CN/DATA-FILES.md 中文文档，同样不支持热部署。
